# Stoperica
Program realizuje digitalnu stopericu u Flowcode okruzenju sa cetiri funkcionalna tastera: Start, Stop, Restart i Memory. Omoguceno je precizno mjerenje vremena, zaustavljanje i ponovno pokretanje brojanja, resetovanje vrijednosti te memorisanje trenutnog izmjerenog vremena radi kasnijeg pregleda.

**#Tehnologija**
-FlowCode
-PIC mikrokontroler
